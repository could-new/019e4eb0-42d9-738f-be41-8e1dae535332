# Bulk Stitching Services App

Yeh app boutiques aur kapray ke brands ke liye bulk stitching (wholesale silai) services ko promote karne ke liye banai gayi hai. Is app ke zariye clients aap ki services dekh sakte hain aur contact kar sakte hain.

## Features
- **Responsive Design**: Mobile, Tablet aur Desktop ke liye mufeed.
- **Service Highlights**: High quality, timely delivery aur wholesale rates ki tafseel.
- **Contact Integration**: Asan rabta karne ke liye contact dialog.

## Tech Stack
- Flutter
- Material 3 Design

## Setup / Run
1. Ensure Flutter is installed.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to start the app.
